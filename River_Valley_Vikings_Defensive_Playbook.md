## 1. DEFENSIVE PHILOSOPHY & IDENTITY

> "We play fast, physical, and disciplined football. Our job is to remove explosive plays, control the box, and force the offense to execute perfectly for an entire drive. We will pressure and change the picture, but never at the cost of leverage, gap integrity, or effort."

### Base Structure

| Element | Detail |
|---------|--------|
| **Personnel** | 4-2-5 base; 3-4 packages (Mint/Ace/Jet/Slip) based on offensive personnel |
| **Strength** | Field ("protect space") |
| **Shell** | Two-high |
| **Box** | 6-man (light box default) |
| **Call Format** | Front + Stunt + Blitz tag(s) + Coverage/Tags |
| **Communication** | Signals with rules/adjustments per formation |

### Non-Negotiables

1. **No explosives.** Make them snap it again. Eliminate busts and cheap yards.
2. **Aggressive with discipline.** Attack without losing leverage or committing penalties.
3. **Box the run.** Set hard edges, compress gaps, keep the ball in help.
4. **Relentless pursuit.** 11 to the ball, correct angles, swarming tackles — finish every snap.
5. **Win leverage.** Outside vs perimeter/QB run; inside vs quick game/crossers.
6. **Win situations.** 1st down sets the series; 3rd down/red zone = get off the field.
7. **Smart aggression.** Pressure to take away something; call off the horses when needed.
8. **Communication standard.** Simple, loud, early. Safeties are fixers. Alerts: BUMP / BANJO / EXCHANGE / UNDER.

### Grading Non-Negotiables

- No lost edges — contain player keeps QB and ball inside
- No uncovered gaps — adjust instantly if gap changes with stunt/pressure
- No free releases — collision/deny access; communicate stack/bunch
- No missed tackles — leverage + near foot + wrap; eliminate YAC
- No penalties that extend drives
- No loafs — pursuit is mandatory on film

## 3. 4-DOWN FRONT CATALOG

### Global Rules (no stunts)

- M fits the open gap to the field side
- W fits the open gap to the boundary side
- Open gap logic: 3-tech closes B → open A; 2/2i/1 closes A → open B
- A and E = contain (set edge, no wrong-arm)
- B = field force. D = boundary force
- 2-back (20/21 personnel): 6-man box default; B is first insert if adding 7th hat
- Puller rule (Power/Counter/Wing-T): Box + overlap; edge outside, M/W inside-out off pull path
- QB run (CAMP): DE (A) is QB player

### TE SET — Default 4-Down Adjustment

Overrides **ALL** fronts **EXCEPT GRIZZLY** when TE/Y-off surface.

- Set front to TE: 3-tech to TE, 2i away
- B inserts into box at LB depth over C gap
- End to TE aligns 7-tech and plays contain
- *Example:* TE to field → A goes to 7-tech on TE, E stays 5 boundary

### Front Alignments

| Front | A | T | N | E | M Fit | W Fit |
|-------|---|---|---|---|-------|-------|
| **SHADE** (base, set field) | 5 | 3 | 2i | 5 | Open A (field) | Open B (boundary) |
| **UNDER** (set boundary) | 5 | 2i | 3 | 5 | Open B (field) | Open A (boundary) |
| **EYES** (balanced) | 5 | 2i | 2i | 5 | Open B | Open B |
| **WIDE** | 5 | 3 | 3 | 5 | Open A | Open A |
| **DEUCES** | 5 | 2 | 2 | 5 | Make T/N right | Make T/N right |
| **GRIZZLY** | 4i | 2i | 2i | 4i | 10-tech; no gap (C to TE) | OLB |
| **BOSS** (bigs field) | 5 | 3(F) | 1(F) | 5 | A gap (boundary) | B gap (boundary) |
| **BOSS UNDER** (bigs bnd) | 5 | 1(B) | 3(B) | 5 | A gap (field) | B gap (field) |

### When to Call

| Front | Best Use |
|-------|----------|
| SHADE | Base vs spread/RPO; stable vs Air Raid |
| UNDER | Tendency breaker; boundary-run weeks; pairs with SLANT |
| EYES | Vs zone/duo; square interior |
| WIDE | Force bounce; vs B-gap heavy / gap schemes |
| DEUCES | Vs Wing-T / pullers; M/W become gap players only if stunt creates it |
| GRIZZLY | Vs power/counter; red zone; short yardage; B and W are OLBs; TE SET does NOT override |
| BOSS | Overload field bigs; pressure setup |
| BOSS UNDER | Overload boundary bigs |

## 4. 3-DOWN PACKAGES (MINT / ACE / JET / SLIP)

### Common Rules

- Field OLB = B. Boundary OLB = E.
- T/N may swap by personnel.
- Slice rule: backside OLB has slicer.
- Strength = Field.

### ILB Fits (Locked)

- **Play-to** (toward strength): STACK B → D (inside-out)
- **Play-away** (away from strength): SLOW PLAY A (patient shuffle, hunt cutback)

### MINT (4i / 0 / 4i)

- **Signal:** "M" on thigh
- A=4i, T=0, N=4i. B/E = OLBs.
- Contain: Man (C0/C1) = B. NINJA/VIKING (MINT only) = M is QB contain.

### ACE (4 / 0 / 4)

- **Signal:** 1 finger up (optional chest tap)
- A=4, T=0, N=4. B/E = OLBs.
- A, T, N **ALL** 2-gapping (no stunt). Contains = OLBs (B and E).

### JET (5 / 0 / 5)

- **Signal:** Whoosh forward (hand flat)
- A=5, T=0, N=5. B/E = OLBs.
- T is 2-gapping. A and N are contain (C-gap edges).

### SLIP (5 / 0 / 4i)

- **Signal:** Mime pushing a wall to the side (two hands)
- A=5, T=0, N=4i. B/E = OLBs.
- 1-gap penetrating. A and E are contain.
- **Use:** 3-down vs spread with NINJA or VIKING behind it. Takes B out of conflict — B plays coverage clean without force/contain responsibilities.

### Contain Matrix

| Package + Coverage | Contain |
|--------------------|---------|
| MINT + Man (C0/C1) | B |
| MINT + NINJA/VIKING | M is QB contain |
| ACE/JET + Man | B |
| ACE/JET + NINJA/VIKING | B contain-first, then match on pass read |
| SLIP + NINJA/VIKING | A and E (B plays coverage clean) |

### TE Adjustment (3-Down)

OLB-only adjust: bump B/E alignments to TE/Y-off surface. Keep Mint/Ace/Jet/Slip interior intact.

## 5. STUNT CATALOG

DL coaching point: read run/pass; on run read get heel depth. Edges play normal unless noted.

### SLANT (to field)

- 4-down: T+N slant to field. 3-down: A+T+N slant.
- **NOT** with Shade. Best fronts: Under, BOSS Under.
- Best pressures: Mike, Will, Bandit, sWarM, BooM, BoW.

### ANGLE (to boundary)

- Same as Slant but to boundary.
- **NOT** with Under/BOSS Under. Best fronts: Shade, BOSS.

### PINCH

- 4-down: T+N pinch A gaps (if already in A, play base). 3-down: A+N pinch B gaps.
- A and E normal. Best fronts: Shade, Under, Wide, Deuces.

### JACKS

- 4-down: T+N shoot B gaps. 3-down: A+N expand to C gaps (NOT pinch).
- A and E contain. **NOT** with Wide. Best fronts: Eyes, Shade, Under. Best pressure: sWarM.

### SPLIT

- With BOSS: 1-tech crosses C face to opposite A. With GRIZZLY: field gap-out.
- A and E contain. Looks weird with BOSS but plays like Shade. Best: Mike, sWarM.

### ANCHOR ATTACK

- A attacks OL, washes them inside (closes one gap in). **NOT** paired with Cobra.

### EDGE ATTACK

- E attacks OL, closes one gap inside. **CAN** pair with Cobra.

### CRASH

- A+E shoot B gaps. T+N shoot A gaps. B/M/W contain.
- **NOT** with Grizzly or Freebird. Best fronts: Wide, Shade, Under. Best pressure: BoW.

### ANCHOR RAVEN (A 5→B)

- A shoots B gap from 5-tech. Signal: point field → flap flap.
- Not with Hammer or staB. If + Bandit = BANDIT RAVEN.

### EDGE RAVEN (E 5→B)

- E shoots B gap from 5-tech. Signal: point boundary → flap flap.

## 7. COVERAGE: NINJA (COVER 7 FAMILY)

Coach calls "NINJA." DBs auto-check. FS owns field checks. D owns boundary checks. Safeties are fixers.

### Alignments

- **Corners (FC/BC):** Inside leverage, 6 yards on #1.
- **Safeties (FS/D):** 10-12 yds deep. Inside foot on #2 (~1 yd inside #2). No #2: poach/post posture, hash-to-middle. Compressed/stack: tighten 1-2 steps, ready to BANJO.
- **B (Bandit):** Apex to field, split between #2 and the box.

### NINJA vs 2x2 — MOD / CLAMP

Two techniques that can be called to **either side.** Default: MOD to field, CLAMP to boundary. CLAMP can also be run to the field. MOD can also be run to the boundary, especially when the ball is in the **middle of the field** (field/boundary distinction is less meaningful).

#### MOD (Sky Principles)

Three-man triangle: Corner + Safety + Flat Player. Corner keys #1. Safety reads #2.

| Position | Key | Rule |
|----------|-----|------|
| **Corner** | #1, reads to #2 | Match #1-to-#2. Stay on top of #1 deep/outside. If #1 breaks short → pass off, zone off. |
| **Safety** | #2, reads to #1 | Match #2-to-#1. #2 vertical → match #2. #2 out/under → work to high hole or rob #1. |
| **Flat Player** | #2, expands to flat | Cover down on #2. Match #2. Push (carry vertical). First man to flat. If flat clears → relate back to hook zone. |

**Default field MOD:** FC = corner, FS = safety, B = flat player.
**Boundary MOD:** BC = corner, D = safety, W = flat player (push to flat).

#### CLAMP (Cloud / Read-2 Principles)

Corner and safety **both read #2.** Clamp viability is a **pre-snap check based on receiver splits** — if #2 is aligned within 5 yards of #1 (tight/compressed), CLAMP works. If #2 is split wider, default to MOD.

| Position | Key | Rule |
|----------|-----|------|
| **Corner** | #2 | #2 out (tight split, 5 yds or less) → **jump #2**, take him in the flat. #2 vertical → stay on #1. #2 inside → stay on #1. |
| **Safety** | #2, reads to #1 | #2 vertical → match #2. #2 out → **flip to #1 deep** (replace corner). #2 inside → carry or pass off #2, bracket #1. |
| **Hook/Curl (#3)** | Late push, reads #2 first | Late push drop to hook/curl. Takes whoever becomes #3. Reads #2 trigger before committing to drop. |

**Default boundary CLAMP:** BC = corner, D = safety, W = hook/curl (#3).
**Field CLAMP:** FC = corner, FS = safety, B = hook/curl (#3).

#### M and W Rules

**M — Primary crosser carrier (all configurations):**

| Read | Rule |
|------|------|
| **Base** | Hook zone. Eyes to QB, relate to underneath threats. |
| **First crosser enters his zone** | Wall it, carry it. Do not hand off. |
| **Crosser carries to boundary** | M carries all the way through. Calls **"CROSS!"** so W knows to stay on his assignment. |
| **Mesh (two crossers)** | Take the deeper / more dangerous crosser. Flat player (B) relates back to pick up the shallow crosser. |

**W — CLAMP hook/curl #3 (default configuration):**

| Read | Rule |
|------|------|
| **Base** | Late push drop to hook/curl. Read #2 trigger, then settle for #3. |
| **Hears "CROSS!" from M** | Stay on #3. M has the crosser. Do not chase. |
| **No #3 threat and crosser enters his zone** | Relate to the crosser. |

**W — MOD flat player (boundary MOD configuration):**

| Read | Rule |
|------|------|
| **Base** | Push to flat. Cover down on #2 boundary. First man to flat. |
| **Flat clears** | Relate back to hook zone. |

#### Crosser Handling Summary

| Situation | M | W | B |
|-----------|---|---|---|
| **No crossers** | Hook zone, relate to threats | CLAMP: hook/curl #3 | MOD: flat player |
| **Single crosser (field → bnd)** | Carries crosser all the way through, calls "CROSS!" | Stays on #3 | Flat clears → relates back to hook |
| **Single crosser (bnd → field)** | Picks up crosser heading field | Stays on #3 | Stays in flat |
| **Mesh (two crossers)** | Takes deeper crosser | Takes #3 (or shallow crosser if #3 not threatening) | Relates back to pick up shallow crosser |

#### Default NINJA 2x2 Alignment

| Side | Technique | Corner | Safety | B / W | M |
|------|-----------|--------|--------|-------|---|
| **Field** | MOD | FC (keys #1) | FS (reads #2-to-#1) | B = flat player | Hook / crosser carrier |
| **Boundary** | CLAMP | BC (keys #2) | D (reads #2-to-#1) | W = hook/curl (#3) | Hook / crosser carrier |

#### When to Flip

| Flip | When | Corner | Safety | B / W | M |
|------|------|--------|--------|-------|---|
| **CLAMP to field** | Field #2 tight split (5 yds or less) | FC (jumps #2 out) | FS (#1 deep if #2 out) | B → hook/curl (#3) | Hook / crosser carrier |
| **MOD to boundary** | Ball in MOF | BC (keys #1 deep) | D (reads #2-to-#1) | W → flat player | Hook / crosser carrier |

### NINJA vs 3x1 — POACH

- **Trips to field: D is the poach safety.**
- **Trips to boundary: FS is the poach safety.**
- #3 vertical: poach player takes it (no free seam).
- #3 shallow/under: poach player communicates and overlaps crossers.
- Backside stays sound — no free go balls or glance RPO.
- Poach player slides post-snap. Pre-snap he is still at 10-12 over #2 in his normal spot.

### NINJA Motion Rule

- **BUMP is for MAN coverages ONLY, not NINJA.**
- Motion creates/removes trips → re-check:
  - 2x2 → 3x1: "POACH! POACH!"
  - 3x1 → 2x2: "MOD/CLAMP!"
  - No trips change: keep original.
- Nearest safety calls re-check; other echoes.

### NINJA Special Formations

- **Bunch/Stack/Tight Splits:** BANJO alert. Corner = first OUT; near safety = first IN; opposite safety = deeper fixer.
- **Rub/Pick:** EXCHANGE automatic. Switch on contact.
- **Empty:** Stay split-field. DBs identify #3 threats.
- **Nub TE/Y-off:** Treat nub as #1. Safety communicates if nub goes vertical.

## 8. COVERAGE: COVER 1 FAMILY

### CAMP Distribution (All 3 Calls)

- FC/BC: man #1 to your side.
- Non-post safeties: #2 to their side, or #3 away if no #2.

### RB Funnel / RAT (All 3 Calls)

- M and W funnel RB.
- RB to your side → take RB (man). RB away → rush.
- **TAMPA** tag: LB away from RB becomes RAT (spot-drop, disrupt crossing lane).

### OREGON (Post = FS)

| Position | Assignment |
|----------|-----------|
| FS | Post/MOF — rotate to middle of field. Fixer — make the corner right. |
| D | Man #2 boundary, or #3 away if no #2 boundary. |
| B | Man #2 field, or #3 away if no #2 field. |
| FC / BC | Man #1 to your side. |
| M / W | RB funnel. RB to you = take. RB away = rush. TAMPA: away LB = RAT. |

### OKLAHOMA (Post = D)

**Pre-snap: FS and D switch sides.** D aligns to field, FS aligns to boundary. This happens before the snap so post-snap rotations are shorter and cleaner.

| Position | Pre-Snap Alignment | Post-Snap Assignment |
|----------|-------------------|---------------------|
| D | Field side (switches from boundary) | Rotate from field to MOF — Post/Fixer. |
| FS | Boundary side (switches from field) | Man #2 boundary, or #3 away. |
| B | Apex (field) | Man #2 field, or #3 away. |
| FC / BC | Normal (inside lev, 6 yds) | Man #1 to your side. |
| M / W | Normal | RB funnel. |

### OHIO (Post = B) — EXCEPTION

| Position | Assignment |

| FS | Man #2 field, or #3 away. |
| D | Man #2 boundary, or #3 away. |
| FC / BC | Man #1 to your side. |
| M / W | RB funnel. |

> **Note:** Ohio is one of two coverages (with Zorro) that changes the pre-snap look.

## 11. PRE-SNAP DISGUISE & ALIGNMENT RULES

### Core Principle

**Every coverage we run looks the same before the snap.** The offense sees the same two-high shell every time. DBs execute their coverage responsibilities post-snap.

> The offense should never know what coverage we are in until after the ball is snapped.

### Exceptions (2)

1. **OHIO:** B moves from apex to ~8 yards deep in the middle of the formation to become the post/MOF safety. His movement tips the coverage.
2. **ZORRO:** B moves from apex to ~8 yards deep in the middle of the formation to key the RB. His alignment change tips the coverage.

### Universal Pre-Snap Alignment

| Position | Pre-Snap Alignment | Key |
|----------|-------------------|-----|
| FC / BC | Inside leverage, 6 yards on #1 | Do not tip coverage with depth or leverage changes |
| FS | 10-12 yds deep, inside foot on #2 (~1 yd inside #2) | If no #2: hash-to-middle, eyes on #3/QB |
| D | 10-12 yds deep, inside foot on #2 (~1 yd inside #2) | If no #2: hash-to-middle, eyes on #3/QB |
| B | Apex to field, split between #2 and box | Same spot every coverage (except Ohio/Zorro) |
| M / W | Standard LB depth over assigned gap | Run-fit alignment does not change by coverage |

### Formation-Driven Adjustments (NOT Coverage-Driven)

- **Compressed/stack splits:** Safeties tighten 1-2 steps, ready for BANJO. Same adjustment in every coverage.
- **Motion:** NINJA re-checks (POACH/MOD/CLAMP). Man coverages use BUMP-BUMP. Either way, the pre-snap shell resets the same.

### Coverage Summary — Pre-Snap vs Post-Snap

| Coverage | Pre-Snap | FS Post-Snap | D Post-Snap | B Post-Snap |
|----------|----------|-------------|-------------|-------------|
| NINJA 2x2 | Two-high | MOD: top-down #2 | CLAMP: control #2 | Apex: #2/#3 threats |

## 12. AUTOS & ALERTS

### BUMP-BUMP (Man Coverages Only — C0/C1)

- Safeties travel. Underneath does NOT. Safeties do NOT cross.
- **BUMP 1** (Safety → Safety): traveling safety hands off because he won't cross.
- **BUMP 2** (Safety → Corner): if motion becomes new widest #1, safety bumps to corner.
- Corner takes ONLY when motion is new widest #1 to that side.

### BANJO (Safety Alert — NOT Signaled In)

- **Trigger:** Stack, tight/close splits, bunch.
- Corner = first OUT. Near safety = first IN. Opposite safety = deeper fixer.

### EXCHANGE (Automatic)

- **Trigger:** Rub/pick route concepts. Switch on contact. Communicate early.

### UNDER (On-Field Alert — Fixer-Driven)

- **Trigger:** Receiver runs UNDER to pick RB-responsible defender AND RB releases fast out.
- **Rule:** RB player takes UNDER (replaces pick). UNDER defender takes RB (swap).
- **Call:** "UNDER! UNDER!" (no numbers). Fixer safety calls it.
- **RB player by coverage:**

| Coverage | RB Player |
|----------|-----------|
| Cover 1 (OR/OK/OH) | M or W |
| Zeus | B or W |
| Zorro | B |
| Zunnel | M or W |
| Zill | W |
| Zike | M |

## 13. RUN-FIT RULES

No explosives. Box it. Relentless pursuit. Force ball to help.

### Base (4-Down, Non-Grizzly)

- **Edges:** A/E contain. **Force:** B = field, D = boundary.

## 14. PASS CONCEPT BREAKDOWN BY COVERAGE

How each coverage defends common pass concepts. Use this section to game-plan coverage calls based on opponent tendencies.

---

### 14.1 FOUR VERTICALS

All four eligible receivers push vertical. Stretches the defense deep, looking for the seam between zones or a 1-on-1 matchup.

#### vs NINJA (Cover 7)
- **FC:** Matches #1 vertical — stay on top, deny explosive.
- **FS:** Top-down on #2 seam. Already in position — no rotation needed.
- **D:** Controls #2 boundary seam. Already in position.
- **BC:** Clamps #1 vertical.
- **B:** Carries #2/#3 vertical threat until FS takes over, then looks for crosser.
- **M/W:** Hook — look for RB leaking into seam.
- **KEY:** Two-high caps all 4 seams. Seams are bracketed. **Best zone look vs 4 verts.**

#### vs OREGON (Cover 1, Post = FS)
- **FC/BC:** Man #1 vertical — stay on top, don't get beat deep.
- **FS:** Post safety → rotates to MOF. Helps whichever side #2 wins on the seam.
- **D:** Man #2 boundary — carry vertical, lean on FS help over the top.
- **B:** Man #2 field — carry vertical, lean on FS help.
- **M/W:** RB funnel.
- **KEY:** Four seams stress the single post. FS has to choose a side.

#### vs OKLAHOMA (Cover 1, Post = D)
- **D:** Starts field pre-snap → rotates to MOF post. Helps on seams.
- **FS:** Starts boundary pre-snap → man #2 boundary seam.
- **B:** Man #2 field seam.
- **FC/BC:** Man #1 vertical.
- **M/W:** RB funnel.
- **KEY:** Same vulnerability as Oregon — single post vs 4 seams.

#### vs ZEUS (Cover 0)
- **FC/BC:** Man #1. Win at the line or lose — no help.
- **FS:** Man #2 field seam. **D:** Man #2 boundary seam. Both carrying vertical.
- **M:** Primary add-on rusher. **B/W:** RB funnel.
- **KEY:** No post safety. If any receiver wins, it's a touchdown. DL must get pressure.

#### vs VIKING (Cover 3)
- **FC/BC:** Bail to deep 1/3. Stay on top of #1 vertical in their zone.
- **FS:** Deep middle 1/3. Splits the seams. Best single-high look vs 4 verts.
- **D:** Seam-curl-flat. Walls #2 boundary vertical, doesn't carry deep.
- **B:** Curl/flat. Walls #2 field vertical, passes off to FC deep.
- **KEY:** Three deep zones cap all four verticals. FS splits both seams in middle 1/3.

---

### 14.2 MESH / CROSSERS

Two receivers cross underneath at 5-8 yards (mesh) or 10-15 yards (drive). Creates pick/rub action and tests zone defenders' ability to pass off.

#### vs NINJA (Cover 7)
- **FC/BC:** Pass off crosser as it leaves zone.
- **FS:** Top-down on #2 vertical.
- **D:** Controls #2 vertical. Walls crosser entering his zone.
- **B:** Walls crosser entering field zone.
- **M/W:** Hook zone — sit in the wash of crossers, collision and disrupt timing.
- **KEY:** Zone pass-offs handle picks naturally. No defender gets screened. **Best look vs mesh.**

#### vs OREGON / OKLAHOMA (Cover 1)
- **FC/BC:** Man on #1 — must fight through the mesh/pick. BANJO or SWITCH helps.
- **Post safety (FS or D):** Reads QB eyes and jumps the crosser QB targets.
- **B:** Man #2 field vertical.
- **D/FS (non-post):** Man #2 boundary vertical.
- **M/W:** If RB releases, take him. Otherwise become RAT.
- **KEY:** Picks/rubs give man defenders trouble. Need aggressive collision at the LOS.

#### vs ZEUS (Cover 0)
- **FC/BC:** Man through the mesh. Must fight picks with no help.
- **FS:** Man #2 field vertical. **D:** Man #2 boundary vertical.
- **M:** Rush. DL pressure must force a quick throw.
- **KEY:** Most vulnerable coverage vs mesh/picks. Need instant pressure.

#### vs VIKING (Cover 3)
- **FC/BC:** Bail deep 1/3. Prevents shots over the top after mesh bait.
- **D/B:** Wall crossers coming through their zone. Don't chase — let them come to you.
- **M/W:** Collision in the hook zone, disrupt timing.
- **KEY:** Zone handles picks. Three deep prevents shots over mesh bait.

---

### 14.3 SMASH (Corner / Hitch)

\#1 runs a hitch at 5-6 yards. #2 runs a corner route behind it. Attacks the flat defender — does he sit on the hitch or carry the corner?

#### vs NINJA (Cover 7)
- **FC:** Matches #1 hitch — sits on it.
- **FS:** Already over #2 — corner route runs right into his coverage. Stays on top.
- **BC:** Clamps #1 hitch.
- **D:** Controls #2 boundary. If corner, carries it.
- **B:** Reads #2 out. If corner, falls under it (hitch help).
- **KEY:** FS already over #2 — corner runs into his coverage. **Strong matchup.**

#### vs OREGON / OKLAHOMA (Cover 1)
- **FC/BC:** Man on #1 hitch — easy, just sit on it.
- **B/D (whoever has #2):** Carries corner route man-to-man.
- **Post safety:** Helps on corner if it goes toward MOF.
- **KEY:** Man handles smash well. Post safety can help on corner.

#### vs ZEUS (Cover 0)
- Man on #1 = hitch is easy. Man on #2 = must carry corner with no help.
- **KEY:** If #2 runs a great corner route, no help over the top.

#### vs VIKING (Cover 3)
- **FC/BC:** Bail to deep 1/3. Corner route runs into their deep zone — stays on top.
- **B/D:** Curl-flat / seam-curl-flat. Sits on the hitch, passes corner to deep 1/3.
- **KEY:** Deep 1/3 caps corner, flat covers hitch. **Best zone look vs smash.**

---

### 14.4 FLOOD / SAIL (3-Level Stretch)

Three receivers to one side at three levels: deep (corner/go), intermediate (dig/out), short (flat/arrow). Stretches two defenders into three zones.

#### vs NINJA (Cover 7)
- **FS:** Takes deepest threat (corner/go) on MOD side.
- **B:** Walls the intermediate route.
- **FC:** Matches #1 if he's running the flat.
- **M:** May need to trigger to the flat if B is occupied.
- **KEY:** The third level (flat) can stress B if he's slow to react.

#### vs OREGON / OKLAHOMA (Cover 1)
- Man assignments handle the levels. Each defender carries his man.
- Post safety leans toward the flood side to help on the deepest route.
- **KEY:** If flood comes from trips, post must commit to one side.

#### vs ZEUS (Cover 0)
- Man coverage — each defender runs with his guy. No flat conflict.
- **KEY:** No deep help on the corner/go route.

#### vs VIKING (Cover 3)
- Deep 1/3 defender caps the corner/go route.
- B or D handles the intermediate. M/W one of them triggers to flat.
- **KEY:** Flat can be vulnerable if underneath dropper is slow or out-leveraged.

---

### 14.5 LEVELS (Hi-Lo Crossers)

Two crossers at different depths (drag at 5 yds, dig at 12 yds). Reads the hook/curl defender — does he drop deep or sit shallow?

#### vs NINJA (Cover 7)
- **FS/D:** Relate to deeper crosser (dig) entering their zone.
- **B/M/W:** Wall the shallow crosser (drag). Don't chase deep.
- **KEY:** Zone reads hi-lo. Defenders play in between. No picks. **Strong matchup.**

#### vs OREGON / OKLAHOMA (Cover 1)
- Man defenders carry their assigned receiver across the field — long chase.
- Post safety reads QB eyes and helps on the deeper crosser.
- **KEY:** Crossing routes create long chases for man defenders.

#### vs ZEUS (Cover 0)
- Man on each crosser with no help. Must stay attached through the route.
- **KEY:** Long crosses + no help = vulnerability if defender trails.

#### vs VIKING (Cover 3)
- **D/B:** Seam-curl-flat — wall the deeper crosser as it enters zone.
- **M/W:** Hook — collision the shallow crosser, disrupt timing.
- **KEY:** Zone layers naturally handle hi-lo. Crossers run into drops.

---

### 14.6 CURL / FLAT

\#1 runs a curl at 12-14 yards. #2 runs a flat/out at 5 yards. Classic 2-man read of the flat defender.

#### vs NINJA (Cover 7)
- **FC:** Sits on curl. **FS:** Over the top of #2, flat route runs into his zone.
- **BC:** Clamps curl. **D:** Controls #2 flat.
- **B:** Reads #2 out to flat, walls it. If vertical, carries.
- **KEY:** Two-high = both curl and flat are covered. No conflict for flat defender.

#### vs OREGON / OKLAHOMA (Cover 1)
- **CB:** Man on #1 curl — sit on it. **B/D:** Man on #2 flat — run with it.
- Post safety looks for deeper threats.
- **KEY:** Man handles curl/flat cleanly.

#### vs ZEUS (Cover 0)
- CB on curl, FS/D on #2 flat. DL must pressure before QB finds the window.

#### vs VIKING (Cover 3)
- **FC/BC:** Bail deep, see curl develop, drive on it from deep 1/3.
- **B/D:** Curl-flat — play flat if #2 goes out, rally to curl if #2 goes vertical.
- **KEY:** Classic Cover 3 conflict. If B/D widens to flat, curl can be open behind him.

---

### 14.7 SLANT / FLAT (Quick Game)

\#1 runs a quick slant. #2 runs flat/arrow. Fast-developing concept that attacks inside leverage.

#### vs NINJA (Cover 7)
- **FC/BC:** Inside leverage pre-snap. Slant comes right to you — collision and match.
- **FS/D:** #2 flat — trigger and drive.
- **KEY:** Inside leverage by corners is the perfect counter to slants.

#### vs OREGON / OKLAHOMA (Cover 1)
- CBs: man on slant. Inside leverage + physicality at the LOS crucial.
- B/D: man on #2 flat — trigger quickly.
- Post safety: read QB, rally to slant if it breaks big.

#### vs ZEUS (Cover 0)
- Man on slant, man on flat. Pressure forces quick throw.
- If CB is physical at the LOS, slant is dead.

#### vs VIKING (Cover 3)
- **FC/BC:** Bail deep — slant breaks underneath. Drive on it immediately.
- **M/W:** Hook — sit on the slant as it comes across. Wall it.
- **B/D:** Curl-flat — trigger to flat.
- **KEY:** Corners bailing deep give up the slant underneath. LBs must be alert.

---

### 14.8 RPO CONCEPTS

QB reads a defender post-snap. If that defender bites on the run fake, QB pulls and throws behind him (bubble, now screen, glance, pop pass).

#### vs NINJA (Cover 7)
- **B** is the key. At apex, reads run/pass. If B triggers to run, #2 screen is open behind him.
- **FS/D:** Pattern-match. If ball goes to #2 quick, react and drive.
- **KEY:** B must not over-commit to the run until he reads the mesh.

#### vs OREGON / OKLAHOMA (Cover 1)
- **B** has #2. If #2 runs a bubble/now screen, B is already on him.
- **KEY:** Man coverage is naturally sound vs RPOs. Each receiver is accounted for.

#### vs ZEUS (Cover 0)
- Man coverage accounts for every receiver. RPO is covered.
- DL pressure forces quick decision. If QB holds it, he gets hit.

#### vs VIKING (Cover 3)
- **B:** Curl-flat — must not vacate flat on run fake. Stay disciplined.
- **D:** Seam-curl-flat — same discipline on boundary side.
- **KEY:** If underneath droppers bite on run fake, quick screens behind them.

---

### 14.9 DOUBLE POST / POST-DIG

Two receivers run post routes (or one post + one dig). Attacks MOF with two threats, stressing single-high safeties.

#### vs NINJA (Cover 7)
- Two-high: each safety matches a post entering their half.
- **KEY:** Double posts are designed to beat single-high. Two-high handles them naturally. **Best coverage vs double post.**

#### vs OREGON / OKLAHOMA (Cover 1)
- Post safety (FS or D) sits in MOF. Two posts converge on him.
- The post he doesn't take is a potential big play.
- **KEY:** Single-high vs double posts is a losing matchup. Need pressure.

#### vs ZEUS (Cover 0)
- Man on both receivers. No deep help. Both posts running free if defenders trail.
- **KEY:** Extremely vulnerable. Must get pressure before routes develop.

#### vs VIKING (Cover 3)
- **FS:** Deep middle 1/3 — splits both posts, takes whichever one QB targets.
- **FC/BC:** Deep 1/3 — lean inside to help with post entering their third.
- **KEY:** Deep middle 1/3 player splits both posts. Better than single-high Cover 1.

---

### 14.10 STICK (Quick Game)

\#1 runs a stick route (6 yds, settle in zone window). #2 runs a flat. #3 or RB pushes vertical as the alert. A 3-level quick concept.

#### vs NINJA (Cover 7)
- **B:** Reads #2 flat, walls it. If #3 pushes vertical, alerts safety.
- **M/W:** Hook zone — sit on the stick route. Don't let #1 settle clean.
- **KEY:** Zone drops cover all three levels.

#### vs OREGON / OKLAHOMA (Cover 1)
- **CB:** Man on #1 stick — sit on it at 6 yards.
- **B/D:** Man on #2 flat. Rally quickly.
- Post safety: read QB, help on the vertical push if it comes.

#### vs ZEUS (Cover 0)
- Man on stick, man on flat, man on vertical. All accounted for.
- Need pressure before QB reads the 3-level progression.

#### vs VIKING (Cover 3)
- **M/W:** Hook — sit on the stick.
- **B/D:** Curl-flat — drive to the flat quickly.
- Deep 1/3: stay over the top of the vertical push.
- **KEY:** Similar to curl/flat — flat defender has a conflict between sitting on stick and driving to flat.

## 20. PRESSURE CARDS (RUSH / DROP ASSIGNMENTS)

Detailed rush/drop assignments for every pressure call. Coverage replacement is **coverage dependent** — the same pressure produces different drop responsibilities based on the coverage called with it.

### Reading the Cards

- **Rushers:** Players who attack the LOS on pass read.
- **Droppers:** Players responsible for coverage zones or man assignments.
- **Coverage replacement:** When a coverage player blitzes, someone else absorbs their responsibility.

## 22. RUN SCHEME SPECIFICS

How we fit each common run scheme. Built from base rules (Section 13). Each scheme identifies: DL responsibility, LB read/fit, edge/force, and key coaching point.

---

### 22.1 INSIDE ZONE (IZ)

**Offense intent:** Combo blocks create movement. RB reads frontside A/B gap, cuts to daylight.

| Position | Responsibility |
|----------|---------------|
| **T** | Play technique. If 3-tech: squeeze B gap, don't get reached. If 2i: close A gap. |
| **N** | Play technique. Hold gap, don't get washed. |
| **A** | Contain. Set edge. Do not get reached — force cutback inside. |
| **E** | Contain. Set edge boundary. |
| **M** | Read guard-to-tackle combo. Fit open gap field side. Downhill, don't drift. |
| **W** | Read guard-to-tackle combo. Fit open gap boundary. **Key:** Be patient — watch for cutback. |
| **B** | Field force. Contain spill from the outside. |
| **D** | Boundary force. |
| **FS/D** | Run support — fill alley on run read. |
| **Key** | IZ is won by interior DL holding gaps and LBs fitting fast. Cutback is the explosive — W must be patient. |

---

### 22.2 OUTSIDE ZONE (OZ)

**Offense intent:** Stretch the defense laterally. RB reads the edge — if it's set, cut back.

| Position | Responsibility |
|----------|---------------|
| **T/N** | Reach/stretch technique. Hold gap, don't get displaced laterally. Fight pressure. |
| **A** | **Critical.** Set hard edge. DO NOT get reached. If edge is set, RB has to cut back into help. |
| **E** | Set hard edge boundary. Same coaching point. |
| **M** | Flow with the play, shuffle and fit. Don't overpursue — stay square. |
| **W** | Shuffle with flow. Be ready for cutback — don't chase frontside. |
| **B** | Force. Attack downhill on the perimeter. Force the cut back inside. |
| **D** | Boundary force on cutback. |
| **Key** | If edges are set, OZ is dead. Ball bounces back into the box. **Pursuit angles** are critical — don't overpursue. |

---

### 22.3 POWER

**Offense intent:** Kick out the edge defender. Pull a guard through the hole. Run behind double team.

| Position | Responsibility |
|----------|---------------|
| **T/N** | Play gap. Hold double team — do not get driven off the ball. |
| **A (play-side)** | Edge player. Can be kicked. If kicked: **wrong-arm the kickout block** (spill the ball outside) OR box (squeeze inside). Follow game plan. |
| **E (backside)** | Chase. Do not let bootleg/keeper come back. |
| **M** | Read pull. Stack behind DL, fit inside-out off pull path. Be physical at the POA. |
| **W** | Read pull. Fit inside-out. If play-away: slow play A, hunt cutback. |
| **B** | Overlap/force outside the kickout. If ball spills, B makes the play. |
| **D** | Boundary force. Overlap if play comes boundary. |
| **Key** | Puller rule: Box + overlap. Edge outside, M/W inside-out off pull path. Collision the puller — don't let him get to the second level clean. |

---

### 22.4 COUNTER

**Offense intent:** Misdirection with two pullers (guard + tackle or guard + H-back). Same blocking scheme as power, but backfield action goes away first.

| Position | Responsibility |
|----------|---------------|
| **T/N** | Play gap. Backside DL: chase through B gap, don't get sealed. |
| **A (play-side)** | Same as Power — edge player, handle kickout. |
| **E (backside)** | **Critical:** Stay home. Do NOT chase the initial backfield flow. Counter boot/keep comes back to the backside. |
| **M** | Read 2nd puller (usually guard). Don't bite on initial flow. Stack and fill. |
| **W** | Slow play. Read 2nd puller. Don't chase initial flow — the play is coming back. |
| **B** | Overlap outside the kick. Same as power. |
| **Key** | Discipline. Do NOT chase initial flow. Read pullers, not backs. Backside E must stay home for the keeper. |

---

### 22.5 DRAW

**Offense intent:** Fake pass, hand off late. Exploits aggressive pass rush.

| Position | Responsibility |
|----------|---------------|
| **T/N** | On pass read: get heel depth (normal rush). When you read draw action (OL stops blocking aggressively, RB takes handoff), **squeeze back down** and close lanes. |
| **A/E** | Contain. Stay in rush lane. Don't get pulled inside — QB might keep. |
| **M** | Pass read → eyes to QB → see the give → trigger downhill fast. Don't hesitate. |
| **W** | Same as M. Read the give, trigger. |
| **B** | Force on draw. Set edge, force RB inside. |
| **Key** | DL discipline on pass rush. Rush with lane integrity — don't all collapse inside. LBs must see the give quickly. |

---

### 22.6 QB RUN / DESIGNED KEEPER (CAMP)

**Offense intent:** QB runs (zone read keep, designed QB power, QB draw).

| Position | Responsibility |
|----------|---------------|
| **A** | **QB player** (CAMP). Keep outside leverage on QB. Do not chase the RB — A has the QB. |
| **E** | Backside contain. If QB cuts back, E makes the play. |
| **T/N** | Play gap responsibility. Don't chase — hold your gap. |
| **M** | Fit off DL. If QB gives to RB, M fills the run gap. If QB keeps, rally to the ball. |
| **W** | Fit off DL. Same read. |
| **B** | Force. If QB gets outside A, B must make the play. |
| **Key** | A is THE player responsible for the QB. He must keep outside leverage and not get caught inside. Everything else plays off A's discipline. |

---

### 22.7 JET SWEEP / SPEED SWEEP

**Offense intent:** Get the ball to the perimeter fast with a jet motion receiver.

| Position | Responsibility |
|----------|---------------|
| **B** | **Primary force player** (field). Must set the edge FAST. Attack downhill — do not let the sweep turn the corner. |
| **D** | Force boundary. If jet comes back (reverse/misdirection), D is the force. |
| **A** | Squeeze inside. Don't chase — hold C gap, force the ball to bounce to B. |
| **E** | Backside: stay home for cutback or reverse. |
| **M** | Fast flow to jet side. Fill inside B's force. |
| **W** | Controlled flow. **Watch misdirection** — jet fakes that come back boundary. |
| **T/N** | Play gap. Don't get reached. Squeeze. |
| **FC** | Contain/force in condensed formations. Otherwise play coverage. |
| **Key** | B must be fast to the edge. Jet sweep is won or lost at the corner. If B sets the edge, the ball cuts back into the box. |

---

## 23. SITUATIONAL DEFENSE

Recommended menus for key game situations. These are **starting points** — adjust weekly based on opponent tendencies.

---

### 23.1 3RD & SHORT (1-3 Yards)

**Philosophy:** Stop the run first. Accept man coverage risk. Be physical at the POA.

| Category | Recommendations |
|----------|----------------|
| **Fronts** | GRIZZLY (primary), WIDE, SHADE |
| **Stunts** | PINCH, CRASH |
| **Pressure** | Mike, sWarM |
| **Coverage** | NINJA (safe), ZEUS (aggressive) |
| **Key Thought** | They want to run it. Line up heavy, stuff the gaps. If they throw, it's likely quick game — inside leverage handles it. |

---

### 23.2 3RD & MEDIUM (4-6 Yards)

**Philosophy:** Disguise and pressure. Two-high shell, attack from unexpected angles.

| Category | Recommendations |
|----------|----------------|
| **Fronts** | SHADE (base), UNDER |
| **Stunts** | SLANT, ANGLE |
| **Pressure** | BooM, BoW, Hammer |
| **Coverage** | NINJA (primary), OREGON |
| **Key Thought** | Most important down in football. Give them a two-high picture, bring pressure from a different spot each time. Mix zone and man behind it. |

---

### 23.3 3RD & LONG (7+ Yards)

**Philosophy:** Rush and cover. Zone coverage to take away quick throws, pressure to collapse the pocket before deeper routes develop.

| Category | Recommendations |
